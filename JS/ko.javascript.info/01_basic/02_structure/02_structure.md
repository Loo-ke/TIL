# 02 structure

## 문
- 어떤 작업을 수행하는 문법 구조와 명령어를 의미한다.
- 코드엔 원하는 만큼 문을 작성할 수 있는데, 서로 다른 문은 세미콜론으로 구분한다.
### 문 구분 에러
```
alert("에러 발생")
[1,2].forEach(alert)
```
- 위의 코드를 실행하면 에러가 발생한다.
```
alert("에러 발생");
[1,2].forEach(alert)
```
- 이 코드를 실행하면 정상 작동!
- 자바스크립트가 대괄호 앞에 세미콜론이 있다고 가정하지 않기 때문에 발생한 에러
- 세미콜론이 없다면 `alert("에러가 발생합니다.")[1, 2].forEach(alert)` 으로 처리됨.
